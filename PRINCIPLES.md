# Principles

## Authority verbs

```
Proof canonizes.
Validation determines.
Attestation witnesses.
Projection displays.
```

## Non-collapse formula

```
Projection is not authority.
Notation is not execution.
Interpretation is not acceptance.
Consensus is not collapse.
```

## Classification precedence

```
classification precedes authority
boundary precedes promotion
distinction precedes counting
```

## Layer model

```
External Computational Environments
        |
    SETCO (transition/classification framework)
        |
    Omi Object Model (pure computational paradigm)
        |
    OMI Artifact Rails (omi-axioms, omi-isa, receipts, projections)
```
