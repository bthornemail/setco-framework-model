# NLP Intake

## Role

SETCO provides the NLP intake surface for the OMI ecosystem. External
natural language, documentation, conversations, and human interpretations
enter through SETCO classification before reaching OMI notation.

## Flow

```
External language (natural language, docs, conversations)
        |
    SETCO NLP Intake (classification)
        |
    OMI-Lisp declaration (structured)
        |
    OMI notation (pure)
```

## Rules

```
NLP proposes. OMI-Lisp declares.
Natural language is not declaration.
Classification precedes authority.
```

## Intake taxonomy

```
rule                     -> classified claim
fact                     -> classified claim
algorithm                -> classified artifact
proof                    -> proof target
proposal                 -> promotion candidate
implementation           -> implementation precedent
projection               -> display only
interpretation           -> teaching surface
artifact                 -> external surface
adapter                  -> boundary surface
application_surface      -> downstream
```

## Source ports

The following corpus ports feed the NLP intake:

```
omi-docs-port    -> doctrine history, transition material
omi-media-port   -> pedagogy, public explanation
omi-vault-port   -> workspace, documentation
```
