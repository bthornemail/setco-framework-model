# State / Phase / Transform

> "SETCO classifies. OMI defines. Projection is not authority."

## Mode Classification

SETCO reads any idea through computational mode as three roles:

### State

What remains stable. The object-position. The identity-bearing checkpoint. In OMI: an address, claim, fact, proof object, register, envelope.

### Transform

What moves or operates. The relation, transition, delta. In OMI: interpolation, validation, promotion, propagation, lowering, compilation, attestation.

### Phase

What orientation or context is visible. The readable surface, classification frame. In OMI: omi-notation, omi-lisp, omi-port, SETCO canvas, blackboard category, proof status, projection surface.

## Mapping to Geometry and Grammar

```text
State      ↔ Node   ↔ Noun
Transform  ↔ Edge   ↔ Verb
Phase      ↔ Face   ↔ Adjective
```

## Boundary

State, transform, and phase are independent classification handles. They are not interchangeable. Conflating them collapses the distinction between what holds, what moves, and what orients.
