# Noun / Verb / Adjective

> "SETCO classifies. OMI defines. Projection is not authority."

## Grammar Classification

SETCO maps ordinary grammar onto its classification handles:

### Noun

What is named. The object. In SETCO: a claim, fact, rule, proof, proposal, artifact, address.

### Verb

What acts, moves, relates, or changes. The operation. In SETCO: a transition, dependency, promotion, propagation, proof step, lowering, transform.

### Adjective

What qualifies or colors it. The qualified surface or context. In SETCO: a context, classification plane, projection, canvas surface, interpretive frame.

## Core Correspondence

```text
Noun      = Node  = State
Verb      = Edge  = Transform
Adjective = Face  = Phase
```

## Purpose

This gives SETCO a universal classification handle before proof, implementation, projection, or interpretation are allowed to act. Any idea can first be asked: is it noun, verb, or adjective? Only after that classification may it proceed.

## Boundary

Grammar is not authority. It is a mnemonic. It prevents category collapse by forcing classification before judgment.
